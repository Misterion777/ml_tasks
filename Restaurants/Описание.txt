Так как заданием предусмотрено предсказать продажи ресторанов исходя из последовательных дат, необходимо использовать методы анализирующие временные ряды.

Изначально были проанализированны исходные данные на предмет отсутствующих значений. Выяснилось, что существуют уникальные рестораны 
с очень маленьким количеством информации за выделенный период (1,2 значений). Соответственно, каждый ресторан был дополнен данными до 100 записей 
(максимум по количеству записей в датасете), основываясь на среднем значении за весь доступный период.
В качестве примера были рассмотрено несколько отдельных ресторанов для декомпозиции их временных рядов на составляющие. С помощью этого легко можно
проследить годовую сезонность ряда (к концу года прибыль растет, а в начале - снижается). Вследствие этого можно ожидать, что должны хорошо себя показать модели, 
учитывающие сезонность.

Для анализа были использованы следующие модели: fbProphet, Exponential Smoothing, ARIMA, SARIMAX.
Модели тестировались на случайном ресторане из исходной выборки.
Для подбора параметров для ARIMA и SARIMAX был использован grid search c оценкой rmse.
Также, в этих двух моделях были учтены exogenous variables - колонки Fuel, IsHoliday, Unemployment, Temperature из исходных данных о городах.

Наилучший результат на выбранном ресторане показал Prophet (RMSE = 3878), вследствие чего эта модель была выбрана для предсказаний финальных результатов.
Также, оказалось, что в тестовой выборке присутствуют рестораны, которых нет в тренировочной, их значения были присвоены средним из предсказанных по городам.
